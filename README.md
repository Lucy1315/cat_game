# 냥이의 별사탕 대작전 🐱⭐

귀여운 고양이 파일럿이 별사탕을 모으며 디저트 군단에 맞서는 아케이드 슈팅 게임.

## 플레이

👉 https://lucy1315.github.io/cat_game/

## 목표

6개 스테이지, 총 30웨이브를 돌파하고 디저트 보스 6마리를 모두 물리치면 클리어!

**스테이지**: 낮 → 노을 → 밤 → 우주 → 사탕나라 → 무지개 피날레 (5웨이브마다 보스전)

## 조작

- 이동: 방향키 / WASD (PC) 또는 손가락 드래그 (모바일)
- 발사: 자동 — 피하는 데만 집중!
- 화면 아래 버튼: ⏸ 일시정지/계속 · ↻ 처음부터 · 🎵 BGM 볼륨 슬라이더
- 단축키: P 일시정지 · M 음소거

## 적

부딪히거나 탄에 맞으면 하트 1개 감소 (하트 5개로 시작, 다 잃으면 게임오버)

- 분홍 젤리: 천천히 흔들흔들 내려옴
- 노란 별: 빠르게 지그재그
- 구름: 단단함(3방) + 물방울 탄을 조준 발사
- 꿀벌: 내려오다가 갑자기 돌진
- 유령 (우주부터): 투명할 땐 총알이 통과 — 나타났을 때 잡기
- 사탕 (사탕나라부터): 잡으면 미니 젤리 2마리로 분열

## 보스

- 푸딩킹 (5·10웨이브): 원형 탄막 + 조준탄
- 도넛대왕 (15·20웨이브): 나선 탄막 추가, 후반엔 꿀벌 소환
- 케이크여왕 (25·30웨이브): 사탕비 + 미니 젤리 부하 소환

## 아이템 (적 처치 시 드롭)

- ⭐ 별사탕: 20개 모으면 냥냥이 분신 등장! 화력 2배 + 피격 시 하트 대신 분신이 희생. 다시 20개 모으면 부활
- 별 모양: 트리플샷 9초 · 세모: 연사 9초 · 동그라미: 실드(1회 방어) · 하트: 생명 +1 (최대 5)
- 💣 폭탄: 화면의 적·탄막 전부 폭발 + 보스 대미지
- 🧲 자석: 10초간 별사탕 자동 흡수

## 팁

- 연속 처치로 콤보 배율 상승 — 콤보 20 이상이면 **피버 모드** (무지개 이펙트 + 배율 최대 x4)
- 피격 시 희생 순서: 실드 → 분신 → 하트. 보스전 전에 실드·분신을 챙겨두세요
- 도넛대왕의 나선탄은 한 방향으로 크게 돌면서 회피
- 맞은 직후 잠깐 무적(깜빡임) — 그 틈에 안전지대로 탈출
- 폭탄은 먹는 즉시 발동 — 화면이 탄막으로 가득할 때 주우면 극적!

## 특징
