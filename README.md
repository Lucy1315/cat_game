# 냥이의 별사탕 대작전

귀여운 고양이 파일럿이 별사탕을 모으며 푸딩킹에 맞서는 아케이드 슈팅 게임.

## 플레이

https://lucy1315.github.io/cat_game/

## 조작

- 이동: 방향키 / WASD / 터치 드래그 · 발사는 자동
- P: 일시정지 · M: 음소거

## 특징

- 15웨이브 x 3스테이지, 5웨이브마다 보스전
- 별사탕 20개를 모으면 냥냥이 분신 등장
- 파워업: 트리플샷 · 연사 · 실드 · 하트
- 콤보 배율, 최고기록 저장, 칩튠 BGM
- 단일 HTML 파일, 모바일 지원
